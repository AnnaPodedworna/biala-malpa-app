# Biała Małpa App

Aplikacja webowa dla Biała Małpa Mexico i Multitap stworzona w Vite + React.

## Funkcje
- Wybór knajpy (Mexico / Multitap)
- Podgląd dań z menu
- Rezerwacja stolika
- Lista 6 piw na Beer Wall (Multitap)
- Wyszukiwarka piw z zewnętrznego API

## Uruchomienie
```bash
npm install
npm run dev
```
